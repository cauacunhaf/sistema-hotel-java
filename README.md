# Sistema de Hotel em Java

Projeto desenvolvido em Java com o objetivo de praticar programação orientada a objetos (POO) e lógica de programação.

## 📌 Descrição

Este projeto simula um sistema simples de gerenciamento de hotel executado no terminal.
O sistema permite realizar operações básicas relacionadas à administração de hóspedes e quartos.

O objetivo do projeto foi praticar conceitos fundamentais da linguagem Java e organização de código utilizando classes e métodos.

## ⚙️ Funcionalidades

O sistema permite:

- Cadastrar hóspedes
- Registrar reservas
- Consultar informações do sistema
- Interagir com o usuário através de um menu no terminal

## 🛠️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- Estruturas de decisão (`if`, `switch`)
- Estruturas de repetição (`while`, `for`)
